# Backend - GoBarber
## Tecnologias utilizadas(até o momento)
- NodeJS
- ExpressJS
- Typescript
- ESLint
- Prettier
- TypeORM
- Docker
- Postgres
- JWT(JSON Web Token)

## Instalando as Dependências
Com o terminal aberto na pasta raíz do diretório da aplicação, basta executar o comando:
```
yarn
```
ou com o npm
```
npm install
```

## Inicializando a Aplicação
Com o terminal aberto na pasta raíz do diretório da aplicação, basta executar o comando:
```
yarn dev:server
```
ou com o npm
```
npm dev:server
```
